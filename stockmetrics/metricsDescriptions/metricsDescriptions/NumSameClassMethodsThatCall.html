Calculates the total number of different methods that call a particular method and that are belong to the same class as this method. Belong to the same class means that called methods are declared in the same class or in the super class or interface of a class in which method for which metric is calculated is declared. A call is counted if it potentially may be a call to method the metric is calculated for.
Abbreviation: SAME_CLASS_METHODS_THAT_CALL
